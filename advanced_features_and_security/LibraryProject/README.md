WELCOME TO DJANGO PROJECT

The project highlights basics of django. The installation process is easy. First we create a viretual environment in which all our project dependencies will be installled usib=ng the command py -m venv .env. The venv command is stored comes already configured in python.

To activate the virtual environment we use the command .env\Scripts\activate. Once the virtual command is activated we install Django and then with startproject command to start the project" django admin startproject LocalLIbrary "

After we start an app called bookshelf that will be the main part of the application with "py manage.py startapp bookshelf'
